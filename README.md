#### Copy vimrc to `~/.vimrc`

#### Install [vim-plug](https://github.com/junegunn/vim-plug)
```
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

Enter vim and run:
```
:PlugUpdate
```

For python:
```
pip install jedi
```
